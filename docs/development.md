# 开发文档

## 仓库初始化
- 使用 Vite 创建 Vue 3 项目，集成 Element Plus 与 Axios
- 配置 `@` 别名，新增 API 封装（`src/api/request.js`, `src/api/audio.js`）
- 重写 `src/App.vue`，实现上传与结果表格播放

## 本地开发
- 启动：`npm run dev`（默认端口 `5173`）
- 后端：`http://localhost:5000`

## 推送流程
1. `git add .`
2. `git commit -m "feat: 更新功能"`
3. `git push`
